# 법정동 CodeBook — 읍면동·리 통합 단위(행정동 포맷 유지)

## 목표

- **행정동 CodeBook과 동일한 컬럼·흐름**(법정동코드·시도명·시군구명·읍면동명·기준연월 등)을 유지한다.
- 법정동 데이터에만 존재할 수 있는 **「리」**는 **별도 리 전용 코드 체계로 관리하지 않는다.**
- 대신 **10자리 법정동코드**를 그대로 두고, **표시·관리 단위(`읍면동명`)**만 **「읍면동」또는「읍면동 + 리명」**으로 확장한다.

## 판별 규칙(코드 10자리)

행정안전부 행정표준코드 관례에 맞춰, **숫자만 추출한 뒤 10자리**를 기준으로 한다.

- `prefix8 = digits[:8]` — 동일 읍면동(상위) 묶음
- `ri2 = digits[8:10]` — 리 구간
  - `ri2`가 비어 있거나 **`00`** → **읍면동 단독** 행(리 없음·또는 리 구간 미사용)
  - 그 외 → **리 단위** 행(통합 표시의 후보)

## 통합(중복 방지) 규칙

같은 `prefix8` 그룹 안에서:

1. **`ri2 != 00` 인 행이 하나도 없으면**  
   - 해당 그룹의 모든 행을 **그대로** 둔다. (`codebook_unit: eup_only`)  
   - 일반적인 **동·읍·면**만 있는 경우, **1건**으로 유지된다.

2. **`ri2 != 00` 인 행이 하나라도 있으면**  
   - **`ri2 == 00` 인 행은 제거**한다.  
     → 요구사항대로 **「읍면동 단독」행은 만들지 않고**, 실제 표현 단위는 **읍면동+리**만 남긴다.
   - 남은 각 리 행에 대해 `읍면동명`을 다음 우선순위로 합성한다.  
     - 제거된 00행의 `읍면동명`을 **부모**로 쓰고, odcloud의 **`리명`**(없으면 `법정리명` 등 별칭)을 **공백 1칸**으로 이어 붙인다.  
     - 부모 행이 없으면(업스트림이 00행을 안 줄 경우) API가 준 `읍면동명`·`리명`만으로 최대한 합성한다.

3. **정렬**  
   - 최종 목록은 **법정동코드(숫자)** 오름차순으로 정렬해 재현 가능하게 한다.

## 구현 위치

- **규칙 모듈**: `server/public_district_codes/codebook_legal_eup_ri.py` — `apply_legal_eup_ri_unification`
- **수집 시**: `run_odcloud_legal_codebook_refresh`에서 SQLite 치환 **직전**에 적용해, DB에는 이미 통합된 `eupmyeondong_name`만 저장한다.
- **조회 시**: `read_odcloud_legal_codebook_snapshot`에서도 동일 적용해, **구버전 DB**(00행+리행이 같이 있던 시기)를 읽어도 응답은 통합 규칙을 만족한다(멱등에 가깝게 동작).

## odcloud 필드

- `_pick_legal_fields` / `normalize_odcloud_row_for_legal_codebook`에서 **`리명`**을 수집한다.  
- 명세 키가 다를 수 있어 **`법정리명`·`리`·`리_명`** 등을 별칭으로 본다.
- 스냅샷·API 응답에서는 **`리명` 컬럼을 노출하지 않고** 제거한다(통합 후 `읍면동명`에만 반영).

## 호환성·장단점 요약

| 관점 | 내용 |
|------|------|
| **행정동 코드북** | 행정동 경로에는 이 변환을 적용하지 않는다. 법정동 GET만 대상이다. |
| **장점** | 리 전용 코드·조인 없이 **한 행 = 한 표시 단위**로 단순화되고, 엑셀·필터·검색이 읍면동명 한 컬럼으로 끝난다. |
| **단점** | DB에 **`리명` 원문을 별도 보존하지 않으면**(현재 스키마) 재통합 시 부모 00행이 없는 불완전 데이터에서는 표시가 API `읍면동명`에 더 의존한다. 완전한 재현이 필요하면 나중에 `리명` 컬럼 저장 확장을 검토한다. |
| **검색·출력** | 사용자 검색어는 **`읍면동명` 통합 문자열** 기준(예: `ㅇㅇ동 a리`)이 가장 안정적이다. |
| **중복 방지** | **「그룹 내 리 행 존재 여부」**로 00행을 제거하는 것이 핵심이다. |
