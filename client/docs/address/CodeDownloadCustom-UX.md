# 코드 다운로드 (Custom) — UI/UX 설계서

## 1. 페이지 전체 컨셉

**한 줄 정의:** 표준 시도 코드(1~17)를 사용자 정의 순서로 재배열하고, 새 코드–시도명 매핑을 즉시 생성·검증·보내는 **오프라인 설정형 도구** 페이지.

**핵심 가치**

- **순서 = 코드**: 리스트 위에서부터 1번이 되므로 직관적.
- **비파괴 비교**: 원래 코드(고정 기준)와 새 코드를 나란히 표시해 혼동 방지.
- **보내기 중심**: 복사 및 TXT/CSV 다운로드로 외부 시스템 연동.

## 2. 기존 「코드 다운로드」와의 관계

| 구분 | 코드 다운로드 (기존) | 코드 다운로드 (Custom) |
|------|----------------------|-------------------------|
| 데이터 | 서버 CodeBook 스냅샷 | 화면에 고정된 17개 시도 |
| 목적 | *attrex 등 스크립트 | 시도 순서·코드 부여만 사용자 정의 |
| 내비 | Address 하위 메뉴 | **동일 레벨 형제** 메뉴 항목 |

## 3. 레이아웃

- **데스크톱**: 좌(약 45%) 드래그 리스트 + 비교 / 우(약 55%) 미리보기 패널.
- **모바일**: 상단 설명 → 리스트 → 미리보기 → 액션 바 세로 스택.

## 4. 주요 섹션

1. 페이지 제목·부제·주의 문구  
2. 시도 순서 (드래그)  
3. 결과 코드 미리보기 (`N:시도명`)  
4. 초기화 / 이 브라우저에 저장 / TXT·CSV 다운로드 / 복사  

## 5. 드래그 리스트

- 행: `[드래그 핸들] [새 코드] 시도명 [기준 코드 칩]`  
- HTML5 DnD + 접근용 위/아래 이동 버튼.

## 6. 미리보기 패널

- 읽기 전용 모노스페이스 블록, 스크롤, 줄 수 표시, 복사 버튼.

## 7. 버튼·상태

- **변경됨** 배지: 현재 순서 ≠ 마지막 저장 기준선.  
- **저장됨**: 저장 직후 피드백(토스트/배너).  
- **초기화**: 표준 1=서울 … 17=세종으로 복귀, 확인 대화상자.

## 8. 사용자 흐름

메뉴 진입 → 설명 확인 → 드래그로 순서 변경 → 미리보기 검토 → 복사 또는 다운로드 → 필요 시 **이 브라우저에 저장** → 초기화로 표준 복귀.

## 9. 와이어프레임 (텍스트)

```
┌─────────────────────────────────────────────────────────────────┐
│ 코드 다운로드 (Custom)                    [변경됨]              │
│ 시도 17개 순서를 바꾸면 위에서부터 1~17이 부여됩니다.            │
├────────────────────────────┬────────────────────────────────────┤
│ 시도 순서 (드래그)         │ 결과 코드 미리보기      17 lines   │
│ [≡] 1 부산…     기준 02    │ 1:부산광역시                       │
│ …                          │ …                                  │
├────────────────────────────┴────────────────────────────────────┤
│ [초기화] [이 브라우저에 저장] [TXT] [CSV]  [복사]               │
└─────────────────────────────────────────────────────────────────┘
```

## 10. UI 문구 예시

- 부제: *표준 시도 코드(1~17) 순서를 바꾼 뒤, 매핑 결과를 복사하거나 파일로 받을 수 있습니다.*
- 주의: *공식 행정 코드와 다를 수 있습니다. 내부 매핑·설문 규칙에만 사용하세요.*

## 11. 실무 UX 포인트

- 원코드·신코드 동시 노출, 미리보기는 읽기 전용, CSV는 UTF-8 BOM 옵션 검토, 저장 의미를 문구로 고정.
